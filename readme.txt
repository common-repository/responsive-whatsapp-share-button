== Description ==

This plugin will add a WhatsApp share button into your every posts when visitor view your site mobile device.

Checkout our other plugins [Click Here](https://www.freewebmentor.com/category/plugins)

== Installation ==

1. Download the archive and unzip it in /wp-content/plugins, upload the archive at Plugins > Add New > Upload Plugin OR install via Plugins > Add New > Search for the Responsive WhatsApp Share Button in the search bar.
2. Activate the plugin through the Plugins menu in WordPress
3. Update the plugin settings under settings at Settings > Responsive WhatsApp Share Button.
4. Click Save!
